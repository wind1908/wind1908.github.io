Расширение радио для браузеров

Здесь доступно расширение для браузеров, которое позволит вам запустить любое радио прямо в браузере максимально быстро

Онлайн радио только с лучшими радиостанциями России и мира. Слушайте популярные русские и зарубежные радиостанции прямо в браузере.

Тысячи онлайн радиостанций с уникальным механизмом подбора радио именно на Ваш вкус. Долгие поиски радиостанций больше не нужны!

Добавляйте любимые станции в закладки и получайте доступ к ним максимально быстро.

Если у Вас будут замечания или идеи по развитию расширения, то пишите их на нашу контактную почту. Все замечания будут учтены, а идеи рассмотрены. Лучшие идеи будут реализованы! Установить радио расширение с официального Chrome Web Store.

Возможности расширения

  • Изменять внешний вид (устанавливать темы) сайта
  • Топ русских радиостанций и ваши избранные станции
  • Список топ русских радио обновляется автоматически
  • Быстрый фильтр поиск радиостанций по избранному и топу
  • Вывод количества потоков у радиостанции на иконке (если их более 1)
  • Глобальный поиск по всем радио (более 2 000 радиостанций)
  • Автоматический выравниватель громкости (Можно задать в настройках)
  • Плагин открывает радио как в новой вкладке, так и в уже существующей вкладке тем самым не дублируя кучу вкладок.
  • Отложенная загрузка изображений (Для более быстрой работы)
  • Возможность максимально быстро обновить список избранного (перелогивание)
  • Совместим с браузерами: Google Chrome, Opera, Яндекс Браузер, Vivaldi, Амиго, Orbitum

Установка расширения в браузеры

Google Chrome, Яндекс Браузер - Установка расширения производится сразу через Chrome Web Store

Opera - Для установки необходимо сначала установить расширение Download Chrome Extension. Это расширение активирует возможность устанавливать расширения через официальный Chrome Web Store. После этого возможно установить его без проблем. Как только расширение будет установлено, его необходимо активировать. Для активации перейдите в список расширений нажав Ctrl + Shift + E и активируйте расширение
